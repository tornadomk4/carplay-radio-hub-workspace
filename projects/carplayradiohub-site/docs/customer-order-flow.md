# Customer Order Flow - Trust-First V1

Goal: make the customer feel like the order is legitimate, vehicle-specific, and safe without forcing Frank to manually answer the same questions forever.

## Recommended Flow

1. Customer lands from Instagram reel, ad, or DM.
2. Page leads with the sale offer:
   - Regular price: $150
   - Sale price: $109.99
   - Backup camera included
   - Fitment checked before fulfillment
3. Customer enters year, make, and model in the fitment checker.
4. Page explains that most standard double DIN vehicles can be supported, but the exact bezel/dash kit/harness depends on the vehicle.
5. Customer submits Jotform order details.
6. Customer pays securely through Stripe using the same name/email as the form.
7. Frank matches the order:
   - Radio
   - Vehicle-specific bezel/dash kit when needed
   - Harness adapter when needed
   - Backup camera
8. Customer receives confirmation and can DM/email with questions.

## Trust Builders To Add

- Real product photo.
- Photo/video of radio installed in Frank's GLI.
- Package contents image.
- Short screen demo.
- "Built by @tornado_mk4" authenticity block.
- Clear 30-day replacement policy.
- Clear same-day shipping language for timely orders.
- Stripe secure checkout language.

## Jotform Fields Needed

- Full name
- Email
- Phone number
- Instagram handle
- Shipping address
- Year
- Make
- Model
- Current radio: factory / aftermarket / unsure
- Installing yourself or using a shop?
- Best contact method
- Notes or photo upload

## Stripe Setup To Confirm

- Product name: CarPlay / Android Auto Radio Kit
- Sale price: $109.99
- Regular price shown on site: $150
- Stripe checkout should use clear customer-facing language.

## Copy Rule

Do not mention Amazon or Prime in customer-facing copy. Public explanation should stay focused on fast fulfillment, fitment-matched kits, secure payment, and support.
