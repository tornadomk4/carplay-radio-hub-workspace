## Upload Instructions (for Frank)

### Redbubble Setup (5 minutes)
1. Go to https://www.redbubble.com/auth/register
2. Create account with tornado.jetta@gmail.com
3. Upload designs as PNG (4500x5400px minimum, transparent background)
4. Set pricing (Redbubble sets default margins — adjust for competitiveness)
5. Fill in title, description, and tags from list above
6. Enable all relevant products for each design

### TeePublic Setup (5 minutes)
1. Go to https://www.teepublic.com/
2. Create account
3. Upload same designs
4. TeePublic handles all fulfillment and customer service

### Etsy + Printful Setup (15 minutes)
1. Create Etsy shop at https://www.etsy.com/sell
2. Connect Printful at https://www.printful.com/
3. Upload designs to Printful, sync to Etsy
4. Etsy brings built-in buyer traffic
