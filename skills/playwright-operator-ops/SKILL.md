# Playwright Operator Ops

## Core Workflow

1. Check whether the Playwright MCP server is available in the active tool list.
2. If it is not available in the current session, verify config with `codex mcp list` and use local Playwright/browser tooling until the host reloads.
3. Keep working artifacts under `C:\Users\frank\.openclaw\workspace\operator-workspace`:
   - `browser-qa/` for flow checks, screenshots, and launch blockers.
   - `market-watch/` for public competitor hooks, ads, offers, and source links.
   - `content-lab/` for scripts, captions, ad concepts, and creative tests.
   - `funnel-ops/` for Stripe/Jotform/site launch checklists.
   - `playwright-mcp/outputs/` for MCP output files.
4. Use browser automation to test the customer path before recommending launch or scale.
5. Save durable decisions and new operating patterns to memory or propose skill updates when they should repeat.

## CarPlay Boundaries

- Research, inspect, draft, test, and organize freely.
- Do not post, comment, DM, run ads, spend money, or make commitments without Frank's approval.
- Keep Amazon/Prime sourcing internal; never add it to customer-facing copy.
- Use confirmed offer terms unless Frank changes them: $109.99 sale price, $150 regular price, backup camera included, 30-day replacement policy, same-day shipping on timely orders, delivery expected in 4 days or less.

## Browser QA Checklist

- Verify desktop and mobile layouts.
- Confirm CTA visibility and link destinations.
- Walk through fitment/order form and payment path.
- Look for overclaims: guaranteed plug-and-play, exact install times, unsupported delivery guarantees, or vague replacement language.
- Capture screenshots or notes for anything that affects conversion, trust, or launch readiness.

## Market Watch Checklist

Capture each useful public example as:

- Source URL.
- Product or offer.
- Hook.
- Visual format.
- CTA.
- Price or discount.
- Comment objections.
- Adaptation for Frank.

Turn each watch session into:

- 5 hooks.
- 3 reel scripts.
- 2 Meta ad angles.
- 3 FAQ/DM reply improvements.
- 1 landing-page copy or offer test.
